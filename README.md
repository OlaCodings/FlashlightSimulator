# FlashlightSimulator

A Unity-based horror game featuring AI pathfinding and dynamic difficulty scaling.

## Features
- AI enemy with pathfinding behavior
- Light-based detection system
- Increasing difficulty as player progresses
- Interactive exploration mechanics

## Tech Used
- Unity (C#)

## What I Learned
- Game AI behavior and pathfinding basics
- Structuring interactive systems
- Debugging and improving performance
